# CSV 导入与导出

如果您在电脑端有已经排好的电子表格（如 Excel 格式的课表），或者教务系统提供了 CSV 导出的接口，您可以通过 SleepIn 提供的 CSV 导入功能批量导入所需课程，省去一条条手动添加的麻烦。

当然，你还需要注意一下 CSV 文件的格式要求，确保它能被正确解析。下面我们来详细介绍一下如何准备 CSV 文件以及导入导出的具体流程。

## 1. 准备 CSV 文件

要使导入能够顺利读取您的数据，所准备的 CSV 必须包含一些**必需字段**（即表头/列名）。SleepIn 默认推荐的栏位如下：

| 列名 | 释义 | 示例 |
|---|---|---|
| 课程名称 | 就是课程名称 (必填) | 线性代数 |
| 教师 | 授课教师 (可选) | 张六 |
| 星期 | 星期几(数字 1-7, 1为周一) (必填) | 4 |
| 开始节次 | 第一节课从第几节开始 (必填) | 1 |
| 结束节次 | 最后一节课到第几节结束 (必填) | 2 |
| 地点 | 上课地点 (可选) | 教二101 |
| 周次 | 需上课的周数区间 (必填) | 1-16|

以下是每列的具体格式要求：

1. `课程名称`
    - 必填，非空字符串。
2. `教师`
    - 可选字符串。
3. `地点`
    - 可选字符串。
4. `星期`
    - 必填整数，范围为 `1..7`。
5. `开始节次`
    - 必填整数，必须在所选课程表的课时安排范围内。
6. `结束节次`
    - 必填整数，必须在课时范围内且 `>= 开始节次`。
7. `周次`
    - 必填字符串，用于描述上课周数。
    - 支持的格式：
      - 区间：`1-16`
      - 自定义列表：`1;3;5;7;8`
      - 奇偶数区间：`1-16(odd)` 或 `1-16(even)`
      - 混合表达式：`1-4;7-9;13;14-18(odd)`
    - 数字和分隔符周围允许留有空格，但输出时建议保持紧凑以提高可读性。

> **注意**：
> - SleepIn 通过列名来识别数据，因此请确保列名与上述表格中的列名完全一致，而不需要注意先后顺序。
> - 请确保 CSV 文件的编码格式为 UTF-8。如使用 Excel 另存为 CSV 时，默认编码可能不是 UTF-8，建议使用文本编辑器打开 CSV 文件并转换编码为 UTF-8 后再导入。

## 2. 导入 CSV 课程表

1. 在 SleepIn 菜单栏中选择“课程表”点击“+”创建新的课程表。
2. 记住先填写好课表的基本信息再点击从 CSV 导入。
3. 点击“保存并从导入 CSV”后，选择你准备好的 CSV 文件。
4. 系统会自动解析 CSV 文件并将课程添加到当前课程表中。

## 3. 导出 CSV 课程表备份

1. 在 SleepIn 菜单栏中选择“课程表”。
2. 点击编辑要导出的课程表。
3. 选择导出 CSV 后选择保存路径即可。
