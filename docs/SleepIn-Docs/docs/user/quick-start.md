# 快速开始

欢迎使用 SleepIn 课程表！这是一款开源的轻量级 Android 课程管理应用，提供清晰、简洁、美观、高效的课程管理体验。本页将引导您快速配置并开始使用。

## 1. 安装与启动

1. 下载 SleepIn 的 ``.apk`` 安装包。
2. 在 Android 设备上安装并打开应用。
3. 开始享受吧！

## 2. 初始配置

初次使用应用，你需要完成以下基础设置：

- **创建作息表**：作息表定义了每天上几节课以及每节课的具体时间。你需要根据自己的学校作息时间来配置哦。
- **创建新课表**：你可以一条一条添加课程，也可以先直接从 [CSV 文件导入](./import-export-csv.md)。

## 3. 开始使用

你可以打开应用来查看课表，当然也可以将小组件添加到桌面来随时查看今天的课程安排。

害怕忘记上课的话，也可以在设置里配置上课提醒，SleepIn 会在上课前通过系统通知提醒你！
