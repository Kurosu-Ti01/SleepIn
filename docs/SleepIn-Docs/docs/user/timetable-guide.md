在快速开始之后，你应该知道 SleepIn 主要就是一个作息表和一个课程表了吧。这里我们来更详细地介绍一下二者使用逻辑。

## 1. 核心概念

1. **课程表 (Timetable)**：一般来说一个学期就有一个课程表。所以通常一个学期对应一个“课程表”。你也就很可能会创建比如“大一上”、“大二下”等多个表。你需要给每个课程表设置一个学期的起始周时间（通常是开学第一周的周一），系统会根据这个时间来计算当前周数以及每节课的具体上课时间。
2. **作息表 (Schedule)**：规定每天上几节课，每节课什么时候上、什么时候下课。每个课程表都需要绑定一个作息表，系统会根据作息表的时间来计算每节课的具体上课时间。你可以创建多个作息表，比如你要区分夏季作息和冬季作息，或者不同学期的作息时间不一样。
3. **课程 (Course)**：就是课程表内具体的条目，你可以设置课程名称、上课地点、授课教师、上课周数等信息。每门课程可以有多段排课记录（如不同周数或不同地点），但它们会被归为同一门课程，方便管理。

## 2. 多周次与多地点

如果你们学校的课程安排比较复杂，比如同一门课在不同周数有不同的上课地点，这很正常，SleepIn 可以很清晰方便地帮你管理这些信息。你只需要在同一门课程下添加多个时间段即可。

## 3. 当前教学周的计算

“当前周”是根据你初始配置中设定的“**开学第一周起止日期**”动态演算出的结果：
- 若你想校正当前周号，不需要修改所有课的配置，仅需修改当前“**课程表**”设定的学期起始周时间。
- 周一始终作为新一周的开始（通常周日作为每周末尾）。

## 4. 多学期切换与归档

如果您拥有多个学期的数据（如不同学年），你可以随时从菜单按钮加载过往学期的课表，而小组件仅显示**当前已激活的课程表**的相关资料。
